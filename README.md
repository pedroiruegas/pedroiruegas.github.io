# Pedro Iruegas — Portafolio

Portafolio personal de Pedro Iruegas, Odoo Developer especializado en Python, OWL, QWeb y personalización de módulos ERP para empresas mexicanas.

## 🌐 Demo

[pedroiruegas.github.io](https://pedroiruegas.github.io)

## 📁 Estructura

```
Portafolio/
├── index.html              # Página principal
├── css/
│   ├── base.css            # Variables, reset, tipografía
│   ├── layout.css          # Split layout (panel izquierdo + derecho)
│   ├── components.css      # Todos los componentes UI
│   └── responsive.css      # Media queries
├── js/
│   ├── theme.js            # Dark / Light mode
│   ├── ui.js               # Loader, scroll progress, nav activo, menú mobile
│   ├── interactions.js     # Contadores animados, reveal on scroll, expandir cards
│   └── cursor.js           # Efecto de glow siguiendo el cursor
└── assets/
    ├── cv_pedro_iruegas.pdf  # CV (agregar manualmente)
    └── images/               # Imágenes de proyectos (opcional)
```

## 🚀 Deploy en GitHub Pages

1. Crea un repositorio llamado `pedroiruegas.github.io`
2. Sube todos los archivos
3. Ve a **Settings → Pages → Source: main / root**
4. Tu portafolio estará en `https://pedroiruegas.github.io`

## ✏️ Personalización

- **Formulario de contacto**: Regístrate en [formspree.io](https://formspree.io), crea un form y reemplaza `YOUR_FORM_ID` en `index.html`
- **CV descargable**: Coloca tu CV en `assets/cv_pedro_iruegas.pdf`
- **Colores**: Edita las variables `--accent` en `css/base.css`

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript vanilla
- Google Fonts (Syne + Space Mono)
- Font Awesome 6
